Using Learn will feel different

We built Learn to feel different than other online learning platform. We wanted it to feel more social, alive, and community-oriented because we believe that online learning should not be a lonely experience. We wanted the content to be super rich and challenging, but also self-improving by virtue of the power of open source. We wanted Learn to emulate the real software development process so that you actually learn how to build real, production software, rather than just syntax or toy apps.

Real Tools

Learn is the only online platform that allows you to use real tools to learn to code. There is no fake "Workspace" feature or over-simplified Web Based Code Editor, you'll be learning to code with the same tools real developers use everyday to build the software that runs the world.

Real Time Chat

When you're stuck, you’ll get help. Right away. No need to wait a week to meet with your mentor. No emails to send. Never feel like you're bugging anyone. Just ask a question, get help. Another Learner just like you, or a coach who has recently been throughthe course, will help get you unstuck.

Learn at Your Own Pace.

Move as fast or as slow as you can. We're going to be here watching you work and making sure you're hitting your goals but you control when and where. You don't need to be online at 9am for a silly “stand up”, you can work when your schedule allows. It's flexible and you're incentivized to finish fast. We'll be on your heels encouraging you and motivating you to stay consistent and dedicated.

Open Curriculum

The base curriculum on Learn is the Web Developer course from the Flatiron School,but that’s just a starting point. We’ve open-sourced the entire track of content (over 300 lessons worth) in the hope that the whole world will help us make it better over time.

As a Learn user, you’ll see an Edit button on every lesson that allows you to update the content of any lesson in-line and submit your change for review by an instructor. Behind the scenes, since all curriculum is hosted on Github, our systems are using the very same git-based workflow that the open-source software development community uses to manage large projects to manage our content editing pipeline. The result for Learners is curriculum that will never be stale, that will be ever-improving and should quickly become the best the world has ever seen.

Transparent Work History via GitHub.

All your work—by which we mean every lesson you’ve ever taken on Learn—ends up on GitHub. Forget about graduating with a handful of projects like some other online coding prog-rams, you're going to graduate with 100s of code lab reposito-ries that show off your coding skills, as well as a bunch of small and large projects that you will build out yourself.
